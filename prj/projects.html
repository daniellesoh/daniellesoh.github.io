Calcifer Trash Bin

For this module project, I plan to make an automatic trash bin which flips open using a infrared sensor, with an acrylic design of Calcifer at the top of the cover; one of the characters in an animation movie called "Howl's Moving Castle". I would like to add a function where the top part of the trash bin will light up after it has detected trash being thrown into the bin.

via GIPHY

Research

With reference to this project, "Automatic Trash Bin", I am thinking of changing the power supply to batteries for better convenience.

For the lights, I plan on using a small LED strip that would light up after sensing motion of the trash going into the bin using reference to this project "Motion-sensing LED hallway"
